Overview
An accordion is a vertically stacked list of headers that reveal or hide associated sections of content. Accordions are useful when you have multiple sections of content that can be collapsed for easier skimming. The accordion component delivers large amounts of content in a small space through progressive disclosure. The user gets key details about the underlying content and can choose to expand that content within the constraints of the accordion. Accordions work especially well on mobile interfaces or whenever vertical space is at a premium.
