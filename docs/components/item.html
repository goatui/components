Events

On click the button a CustomEvent p4Click is triggered by the element.

Click me Disabled Button 
                    document.querySelector('#click-me-btn').addEventListener('p4Click', function(event) {
    alert("Button clicked");
});

document.querySelector('#disabled-btn').addEventListener('p4Click', function(event) {
    alert("Button clicked"); // this will not be fired
});
