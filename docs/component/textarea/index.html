Actions

Make input look inactive by adding the disabled boolean attribute to any <input> element.

{% highlight js %} document.querySelector('#input-with-actions').actions = [{ icon: 'search', eventName: 'search-item' }, { icon: 'x', eventName: 'close' }, { icon: 'info-circle', eventName: 'info-circle', disabled: true }]; document.querySelector('#input-with-actions').addEventListener('goat:action-click', function(evt) { console.log("Action clicked :: " + evt.detail.eventName); }); {% endhighlight %}

Events

On input change a CustomEvent goat:change is triggered by the element.

{% highlight js %} document.querySelector('#name-input-field').addEventListener('goat:input', function(event) { console.log("goat:input :: " + event.target.value); }); document.querySelector('#name-input-field').addEventListener('goat:change', function(event) { console.log("goat:change :: " + event.target.value); }); {% endhighlight %}
