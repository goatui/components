Sizes

Fancy larger or smaller input? Add size attribute for additional sizes.

                  <p4-input size="lg" label="Large Input"></p4-input>
                  <p4-input label="Medium Input"></p4-input>
                  <p4-input size="sm" label="small Input"></p4-input>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                    <p4-input disabled>Disabled Button</p4-input>
                    <p4-input variant="dashed" disabled="true">Disabled Dashed Button</p4-input>
                    <p4-input variant="link" disabled="true">Disabled link</p4-input>

Events

On input change a CustomEvent p4Change is triggered by the element.

Output :: 
                    <script>
 document.querySelector('#name-input-field').addEventListener('p4Change', function(event) {
      document.getElementById('change-output').innerText = event.target.value;
});
</script>
