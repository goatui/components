Actions

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                   <p4-input id='input-with-actions'></p4-input>
          <script>
            document.querySelector('#input-with-actions').actions = [{
              icon: 'search',
              eventName: 'search-item'
            }, {
              icon: 'x',
              eventName: 'close'
            }, {
              icon: 'info-circle',
              eventName: 'info-circle',
              disabled: true
            }];
            document.querySelector('#input-with-actions').addEventListener('p4:action-click', function(evt) {
              console.log("Action clicked :: " + evt.detail.eventName);
            })
          </script>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                  <p4-input disabled>Disabled Button</p4-input>
<p4-input disabled value='With Value'>Disabled Button</p4-input>

Events

On input change a CustomEvent p4:change is triggered by the element.

                    <script>
document.querySelector('#name-input-field').addEventListener('p4:input', function(event) {
       console.log("p4:input :: " + event.target.value);
});
document.querySelector('#name-input-field').addEventListener('p4:change', function(event) {
      console.log("p4:change :: " + event.target.value);
});
</script>
