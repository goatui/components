Use P4RM’s custom input component for capturing input in forms, dialogs, and more with support for multiple sizes, variants and more.

Usage

Script tag

  • Put a script tag similar to this <script type="module" src="https://unpkg.com/@p4rm/ui@{{site.project_version}}/dist/p4rm-ui/p4rm-ui.esm.js"></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!-- Default Input -->
<p4-input placeholder="Name"></p4-input>

<!--  Input with value -->
<p4-input value="John Snow"></p4-input>

<!-- Input with placeholder -->
<p4-input placeholder="Enter Input"></p4-input>

<!-- Input with clear button when there is a value -->
<p4-input clear-input value="clear me"></p4-input>

Examples

Here are several predefined input styles, each serving its own semantic purpose.

                 <p4-input label="Name" placeholder="Person Name"></p4-input>
                 <p4-input label="Name" placeholder="Person Name" value="Ravi"></p4-input>
                 <p4-input variant="dashed" label="Name" placeholder="Person Name"></p4-input>

Assign value

                    <script>
document.querySelector('#name-field').value = "John Snow";
</script>

Inline

Display label and field on same line.

                  <p4-input inline label="Address" value="Hyderabad, Telangana, India"></p4-input>

Sizes

Fancy larger or smaller input? Add size attribute for additional sizes.

                  <p4-input size="lg" label="Large Input"></p4-input>
                  <p4-input label="Medium Input"></p4-input>
                  <p4-input size="sm" label="small Input"></p4-input>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                    <p4-input disabled>Disabled Button</p4-input>
                    <p4-input variant="dashed" disabled="true">Disabled Dashed Button</p4-input>
                    <p4-input variant="link" disabled="true">Disabled link</p4-input>

Events

On click the button a CustomEvent buttonClick is triggered by the element.

Note: Check browser console for output.

                    <script>
document.querySelector('#name-input-field').addEventListener('inputChange', function(event) {
    console.log('Input changed to :: ' + event.detail.newValue + ' - from :: ' + event.detail.oldValue);
});
</script>
