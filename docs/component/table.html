{{ page.title }}

Use Idli’s table component for displaying information in a way that’s easy to scan, so that users can look for patterns and insights. They can be embedded in primary content, such as cards.

Usage

Script tag

  • Put a script tag similar to this <script src='https://unpkg.com/@idli/idli-table@{{ page.version }}/dist/idli-table.js'></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>{{ page.title }}</title>
  <script src='https://unpkg.com/@idli/idli-table@{{ page.version }}/dist/idli-table.js'></script>
</head>
<body>
   <idli-table id="table"
              selection-type="checkbox"
              row-key="_id"
              column-config='[{"name":"name","label":"Name","width":300,"fixed":true}{"name":"company","label":"Company","width":500}]'
              data='[{"_id":"5e7118ddce4b3d577956457f", "name":"Courtney","company":"XYQAG"},{"_id":"5e7118dd240daa9a2f209fd9","company":"STRALOY"}]'></idli-table>
</body>
</html>
