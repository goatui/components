Events

On click the button a CustomEvent p4:click is triggered by the element.

                    document.querySelector('#click-me-btn').addEventListener('p4:click', function(event) {
    alert("Button clicked");
});

document.querySelector('#disabled-btn').addEventListener('p4:click', function(event) {
    alert("Button clicked"); // this will not be fired
});
