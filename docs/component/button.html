Disabled state

Make buttons look inactive by adding the disabled boolean attribute to any <button> element.

Disabled Button Disabled Secondary Button Disabled Ghost Button 
                <p4-button disabled>Disabled Button</p4-button>
 <p4-button type="secondary" disabled="true">Disabled Secondary Button</p4-button>
 <p4-button type="ghost" disabled="true">Disabled Ghost Button</p4-button>

Events

On click the button a CustomEvent p4Click is triggered by the element.

Click me Disabled Button 
                    document.querySelector('#click-me-btn').addEventListener('p4Click', function(event) {
    alert("Button clicked");
});

document.querySelector('#disabled-btn').addEventListener('p4Click', function(event) {
    alert("Button clicked"); // this will not be fired
});
