{{ page.title }}

Use P4RM’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Usage 

                
<script type="module" src="https://unpkg.com/@p4rm/ui@{{site.project_version}}/dist/p4rm-ui/p4rm-ui.esm.js"></script>

<!-- Default -->
<p4-button size="lg" variant="primary">Click me</p4-button>

Variants

Here are several predefined button styles, each serving its own semantic purpose.

Default Primary Danger Dashed Link 
                 <p4-button>Default</p4-button>
                 <p4-button variant="primary">Primary</p4-button>
                 <p4-button variant="danger">Danger</p4-button>
                 <p4-button variant="dashed">Dashed</p4-button>
                 <p4-button variant="link">Link</p4-button>

Sizes

Fancy larger or smaller buttons? Add size attribute for additional sizes.

Large Button Medium Button Small Button 
                <p4-button size="lg">Large Button</p4-button>
<p4-button>Medium Button</p4-button>
<p4-button size="sm">Small Button</p4-button>

Disabled state

Make buttons look inactive by adding the disabled boolean attribute to any <button> element.

Disabled Button Disabled Dashed Button Disabled Link 
                <p4-button disabled>Disabled Button</p4-button>
 <p4-button variant="dashed" disabled="true">Disabled Dashed Button</p4-button>
 <p4-button variant="link" disabled="true">Disabled link</p4-button>

Events

On click the button a CustomEvent p4Click is triggered by the element.

Click me Disabled Button 
                    document.querySelector('#click-me-btn').addEventListener('p4Click', function(event) {
    alert("Button clicked");
});

document.querySelector('#disabled-btn').addEventListener('p4Click', function(event) {
    alert("Button clicked"); // this will not be fired
});
