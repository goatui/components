{{ page.title }}

Use Idli’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Usage

Script tag

  • Put a script tag similar to this <script type="module" src="https://unpkg.com/@p4rm/ui@{{site.project_version}}/dist/p4rm-ui/p4rm-ui.esm.js"></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>{{ page.title }}</title>
  <script type="module" src="https://unpkg.com/@p4rm/ui@{{site.project_version}}/dist/p4rm-ui/p4rm-ui.esm.js"></script>
</head>
<body>
  <p4-icon size="lg" variant="primary">Click me</p4-icon>
</body>
</html>

Disabled state

Make buttons look inactive by adding the disabled boolean attribute to any <button> element.

Disabled Button Disabled Dashed Button Disabled Link 
                    <p4-icon disabled>Disabled Button</p4-icon>
                    <p4-icon variant="dashed" disabled="true">Disabled Dashed Button</p4-icon>
                    <p4-icon variant="link" disabled="true">Disabled link</p4-icon>

Events

On click the button a CustomEvent buttonClick is triggered by the element.

Click me Disabled Button 
                    <script>
document.querySelector('#click-me-btn').addEventListener('buttonClick', function(event) {
    alert("Button clicked");
});
document.querySelector('#disabled-btn').addEventListener('buttonClick', function(event) {
    alert("Button clicked"); // this will not be fired
});
</script>
