{{ page.title }}

Use Idli’s custom checkbox component for capturing input in forms, dialogs, and more with support for multiple sizes, variants and more.

Usage

Script tag

  • Put a script tag similar to this <script src='https://unpkg.com/@idli/{{ page.github_project }}@{{ page.version }}/dist/idli-checkbox.js'></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>{{ page.title }}</title>
  <script src='https://unpkg.com/@idli/{{ page.github_project }}@{{ page.version }}/dist/idli-checkbox.js'></script>
</head>
<body>
   <idli-checkbox label="Name"></idli-checkbox>
   <idli-checkbox variant="dashed" label="Name"></idli-checkbox>
</body>
</html>

Examples

Here are several predefined checkbox styles, each serving its own semantic purpose.



                 <idli-checkbox label="Name" value="true"></idli-checkbox>
                 <idli-checkbox variant="dashed" label="Name" placeholder="Person Name"></idli-checkbox>

Script Get/Set value

                    <script>
document.querySelector('#ice-cream-field').value = true;
console.log(document.querySelector('#ice-cream-field').value); //get value
</script>

Inline

Display label and field on same line.

                  <idli-checkbox inline label="Want a ice cream?"></idli-checkbox>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                    <idli-button disabled>Disabled Button</idli-button>
                    <idli-button variant="dashed" disabled="true">Disabled Dashed Button</idli-button>

Events

On click the button a CustomEvent buttonClick is triggered by the element.

Note: Check browser console for output.

                    <script>
document.querySelector('#change-checkbox-field').addEventListener('inputChange', function(event) {
    console.log('Input changed to :: ' + event.detail.newValue + ' - from :: ' + event.detail.oldValue);
});
</script>
