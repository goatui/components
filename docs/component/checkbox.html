{{ page.title }}

Use Idli’s custom checkbox component for capturing input in forms, dialogs, and more with support for multiple sizes, variants and more.

Usage 

                
<script type="module" src="https://unpkg.com/@p4rm/ui@{{site.project_version}}/dist/p4rm-ui/p4rm-ui.esm.js"></script>

<!-- Default -->
<p4-checkbox label="Name"></p4-checkbox>
<p4-checkbox variant="dashed" label="Name"></p4-checkbox>

Examples

Here are several predefined checkbox styles, each serving its own semantic purpose.



                 <p4-checkbox label="Name" value="true"></p4-checkbox>
                 <p4-checkbox variant="dashed" label="Name" placeholder="Person Name"></p4-checkbox>

Script Get/Set value

                    <script>
document.querySelector('#ice-cream-field').value = true;
console.log(document.querySelector('#ice-cream-field').value); //get value
</script>

Inline

Display label and field on same line.

                  <p4-checkbox inline label="Want a ice cream?"></p4-checkbox>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                    <idli-button disabled>Disabled Button</idli-button>
                    <idli-button variant="dashed" disabled="true">Disabled Dashed Button</idli-button>

Events

On click the button a CustomEvent buttonClick is triggered by the element.

Note: Check browser console for output.

                    
document.querySelector('#change-checkbox-field').addEventListener('p4Change', function(event) {
    console.log('Input changed to :: ' + event.detail.value);
});
