Use Idli’s custom script editor component for capturing input in forms, dialogs, and more. Built on monaco.

Examples

Here is script editor component, currently supports JS and JSON editing.

document.getElementById('business-rule-editor').value = `/**

Disabled state

Make editor look inactive by adding the disabled boolean attribute to any <input> element.

Please click on checkbox and try disabling able editor.

                    <p4-script-editor disabled></p4-script-editor>
