{{ page.title }}

Use Idli’s custom script editor component for capturing input in forms, dialogs, and more. Built on monaco.

Usage

Script tag

  • Put a script tag similar to this <script src='https://unpkg.com/@idli/{{ page.github_project }}@{{ page.version }}/dist/p4-script-editor.js'></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>{{ page.title }}</title>
  <script src='https://unpkg.com/@idli/{{ page.github_project }}@{{ page.version }}/dist/p4-script-editor.js'></script>
</head>
<body>
   <p4-script-editor id="business-rule-editor"></p4-script-editor>
</body>
</html>

document.getElementById('business-rule-editor').value = `function myBusinessRuleFunction(){
    const a = 1;

Disabled state

Make editor look inactive by adding the disabled boolean attribute to any <input> element.

Please click on checkbox and try disabling able editor.

                    <p4-script-editor disabled></p4-script-editor>
