{{ page.title }}

Use Idli’s custom input component for capturing input in forms, dialogs, and more with support for multiple sizes, variants and more.

Usage

Script tag

  • Put a script tag similar to this <script src='https://unpkg.com/@idli/{{ page.github_project }}@{{ page.version }}/dist/idli-number.js'></script> in the head of your index.html
  • Then you can use the element anywhere in your template, JSX, html etc 
                <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>{{ page.title }}</title>
  <script src='https://unpkg.com/@idli/{{ page.github_project }}@{{ page.version }}/dist/idli-number.js'></script>
</head>
<body>
   <idli-number label="Name" placeholder="John Snow"></idli-number>
   <idli-number variant="dashed" label="Name" placeholder="John Snow"></idli-number>
</body>
</html>

Examples

Here are several predefined input styles, each serving its own semantic purpose.

                 <idli-number label="Roll Number" placeholder="Enter Roll No."></idli-number>
                 <idli-number label="Salary" placeholder="Salary" value="Ravi"></idli-number>
                 <idli-number variant="dashed" label="Name" placeholder="Person Name"></idli-number>

Assign value

                    <script>
document.querySelector('#roll-number').value = 200;
</script>

Inline

Display label and field on same line.

                  <idli-number inline label="Street Number" value="1230"></idli-number>

Sizes

Fancy larger or smaller input? Add size attribute for additional sizes.

                  <idli-number size="lg" label="Large Input"></idli-number>
                  <idli-number label="Medium Input"></idli-number>
                  <idli-number size="sm" label="small Input"></idli-number>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                    <idli-number disabled>Disabled Button</idli-number>
                    <idli-number variant="dashed" disabled="true">Disabled Dashed Button</idli-number>
                    <idli-number variant="link" disabled="true">Disabled link</idli-number>

Events

On click the button a CustomEvent buttonClick is triggered by the element.

Note: Check browser console for output.

                    <script>
document.querySelector('#serial-no-field').addEventListener('inputChange', function(event) {
    console.log('Input changed to :: ' + event.detail.newValue + ' - from :: ' + event.detail.oldValue);
});
</script>
