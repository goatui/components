Variants

Here are several predefined input styles, each serving its own semantic purpose.

                 <goat-textarea label="Name" placeholder="Person Name"></goat-textarea>
                 <goat-textarea variant="dashed" value="Some Value"></goat-textarea>

Sizes

Fancy larger or smaller input? Add size attribute for additional sizes.

                  <goat-textarea size="lg" placeholder="Large"></goat-textarea>
                  <goat-textarea placeholder="Medium"></goat-textarea>
                  <goat-textarea size="sm" placeholder="Small"></goat-textarea>

Actions

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                   <goat-textarea id='input-with-actions'></goat-textarea>
          <script>
            document.querySelector('#input-with-actions').actions = [{
              icon: 'search',
              eventName: 'search-item'
            }, {
              icon: 'x',
              eventName: 'close'
            }, {
              icon: 'info-circle',
              eventName: 'info-circle',
              disabled: true
            }];
            document.querySelector('#input-with-actions').addEventListener('goat:action-click', function(evt) {
              console.log("Action clicked :: " + evt.detail.eventName);
            })
          </script>

Disabled state

Make input look inactive by adding the disabled boolean attribute to any <input> element.

                  <goat-textarea disabled>Disabled Button</goat-textarea>
<goat-textarea disabled value='With Value'>Disabled Button</goat-textarea>

Events

On input change a CustomEvent goat:change is triggered by the element.

                    <script>
document.querySelector('#name-input-field').addEventListener('goat:input', function(event) {
       console.log("goat:input :: " + event.target.value);
});
document.querySelector('#name-input-field').addEventListener('goat:change', function(event) {
      console.log("goat:change :: " + event.target.value);
});
</script>
